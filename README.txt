# 📊 Amazon Sales Analysis  

This project analyzes **Amazon sales data** using **Python in Google Colab**.  
It includes **data cleaning, sales trends, dashboards, and KPIs** to identify key insights such as:  
- Top-selling product categories  
- Most profitable regions  
- Monthly sales performance  
- Overall sales and profit KPIs  

---

## 📂 Project Structure
```
├── amazon_sales.csv        # Cleaned dataset
├── Amazon_Sales_Analysis.ipynb   # Google Colab Notebook
├── README.md               # Project documentation
```

---

## ⚙️ Tools & Libraries
- **Python** (Pandas, Matplotlib, Seaborn, Plotly)  
- **Google Colab** (for data visualization & analysis)  
- **Excel/Power BI** (optional, for additional dashboards)  

---

## 🚀 Features
✅ Data Cleaning & Preparation  
✅ Sales Trend Analysis (Monthly/Yearly)  
✅ Top-Selling Products Visualization  
✅ Profitability by Region  
✅ KPI Dashboard (Total Sales, Profit, Best Product, Best Region)  
✅ Interactive Geographical Map (using Plotly)

## ▶️ How to Run
1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/amazon-sales-analysis.git
   ```
2. Open the notebook in **Google Colab** or **Jupyter Notebook**:
   ```bash
   Amazon_Sales_Analysis.ipynb
   ```
3. Install dependencies (if needed):
   ```bash
   pip install pandas matplotlib seaborn plotly
   ```
4. Run all cells to generate the analysis & visuals.
